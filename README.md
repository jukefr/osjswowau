# osjswowau
open source javascript wow addon updater

## supported
- curse
- elvui + elvui addons

## usage (currently not published, see #dev)
```bash
$ npx osjswowau
```

## dev
```bash
$ git clone ... && cd ...
$ npx .
```

## debug
you can toggle `debug` to `true` in the config to get more information if something goes wrong and for bug reports.
